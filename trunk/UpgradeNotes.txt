****************************************************************
**  Upgrading from InsiteCreation 2006 or InsiteCreation 2007  **
****************************************************************

IMPORTANT:
This new version contains major enhancements and is not fully compatible with 
previous versions (InsiteCreation 2006 & InsiteCreation 2007): 
- Image Gallery images will not be visible after upgrading.
  You need to re-create image gallery pages.
- Events pages will be converted to regular (Calendar-based) news listing.

1. Copy (Replace) these files on your server with the new files provided in the new package:
    - All files in folder resources/photos
	- All files in folder App_Code
	- All files in folder App_LocalResources
	- All files in folder App_Themes
	- All files in folder Bin
	- All files in folder dialogs
	- All files in folder modules
	- All files in folder systems
	- All files in folder templates
	- default.aspx
	- default.aspx.vb

2. Run setup/storeproc.sql against your database using Query Analyzer or SQL Server Management Studio.

3. Add this line within <appSettings> in web.config:

	<add key="UseSecureFileStorageForViewing" value="no"/>
	<add key="Shop" value="no"/>

4. In default.master and print.master of your templates, please change:

	<form id="form1" runat="server">
	...
	</form>

with:

	<frm:Form id="Form1" runat="server">
	...
	</frm:Form>

5. In default.master and print.master of your templates, please add the following new placeholders:

	<asp:ContentPlaceHolder ID="placeholderFileView" runat="server">
	</asp:ContentPlaceHolder>   

	<asp:ContentPlaceHolder ID="placeholderFileDownload" runat="server">
	</asp:ContentPlaceHolder> 

	<asp:ContentPlaceHolder ID="placeholderListing" runat="server">
	</asp:ContentPlaceHolder> 

	<asp:ContentPlaceHolder ID="placeholderArchives" runat="server">
	</asp:ContentPlaceHolder>   

	<asp:ContentPlaceHolder ID="placeholderCategoryList" runat="server">
	</asp:ContentPlaceHolder>   

	<asp:ContentPlaceHolder ID="placeholderSubscribe" runat="server">
	</asp:ContentPlaceHolder>

	<asp:ContentPlaceHolder ID="placeholderScript" runat="server">
	</asp:ContentPlaceHolder>  

Note: please see included templates for examples.

6. If you're using "placeholderLogin" in your templates:

    <asp:ContentPlaceHolder ID="placeholderLogin" runat="server"></asp:ContentPlaceHolder> 
    
Please replace it with:

    <asp:ContentPlaceHolder ID="placeholderRegister" runat="server"></asp:ContentPlaceHolder> 
    <asp:ContentPlaceHolder ID="placeholderWelcome" runat="server"></asp:ContentPlaceHolder> 
    <asp:ContentPlaceHolder ID="placeholderAdminWorkspaceLink" runat="server"></asp:ContentPlaceHolder> 
    <asp:ContentPlaceHolder ID="placeholderLoginLogout" runat="server"></asp:ContentPlaceHolder> 

If you need to hide Register Link, just remove the placeholder "placeholderRegister".
If you need to hide Login/Logout link, just remove the placeholder "placeholderLoginLogout".
If you need to hide just the Login Link, please use "placeholderLogout" instead of "placeholderLoginLogout".

    <asp:ContentPlaceHolder ID="placeholderLogout" runat="server"></asp:ContentPlaceHolder> 

7. Add this line within <pages><controls> (in web.config):

	<add tagPrefix="frm" namespace="FormReplace" assembly="App_Code"/>
